## 1. 分析方法

### 抽帧策略
- **均匀抽帧**：`ffmpeg -vf fps=1/5`，每 5 秒一帧，共 107 帧，存入 `/tmp/style_frames_all/uniform_*.jpg`
- **场景切换密集抽帧**：`ffmpeg -vf select='gt(scene,0.3)'`，共 161 个关键帧，存入 `/tmp/style_frames_all/scene_*.jpg`
- **代表性帧精选**：从 107 均匀帧中每 3 帧取 1，共 35 帧复制到 `eval/style_frames/`，覆盖全片每 15 秒一个样本点
- **逐帧视觉分析**：用 Read 工具逐批查看所有 35 帧，分类标注

### 音频分析
- `ffmpeg` 提取 16kHz 单声道 WAV
- faster-whisper（base 模型）转写前 120 秒，得到 665 字符
- 推算全片语速：**约 332 字/分钟**

## 2. 关键视觉发现

### 2.1 整体结构

全片采用典型「科普解说」竖屏格式，结构分为：

| 段落类型 | 占比估计 | 描述 |
|---|---|---|
| 主播主镜头（talking head） | ~57% | 固定中近景，户外咖啡座，自然光，背景虚化 |
| B-roll + PiP | ~20% | 主播缩小为右上/左上角子画面，B-roll 全屏 |
| 数据可视化 + PiP | ~11% | 图表/截图全屏，主播 PiP 同上 |
| 关键词动画覆盖 | ~7% | 大字关键词叠加在主播镜头上（半屏/全屏） |
| 主题全屏卡（话题 badge / 主题背景） | ~5% | 章节过渡时，主播以圆形或全卡呈现 |

### 2.2 字幕

| 属性 | 观测值 |
|---|---|
| 是否双语 | **否** — 仅中文字幕，无英文逐句字幕 |
| 位置 | 画面下方约 70–82% 处（Y ≈ 900–1050px，1280px 高度）|
| 字色 | 纯白，带黑色描边/阴影（高对比度，任何背景可读） |
| 单屏字数 | 4–16 字，多数 6–12 字（1–2 个短句/屏） |
| 字体风格 | 无衬线黑体类，中等字重，无动态出字效果（静态显示） |
| 特殊字幕 | 关键词 badge 内的「中文主标题 + 英文副标题」为装饰性双语，非逐句字幕 |

### 2.3 数据可视化

在 35 帧样本中，出现数据可视化的帧共 4 帧（帧号 0046, 0052, 0070, 0088），推算全片约 4–6 次数据可视化切入：

| 出现时间（估计） | 类型 | 描述 |
|---|---|---|
| ~3:45–4:00 | 大数字 pill badge | CPI 1.0%、PPI 0.5%，圆角黑色/黄色 pill，叠加在新闻截图上；字号极大，3 秒内可读 |
| ~4:15–4:30 | 柱状图（截图引用） | 中国内地个人奢侈品市场销售额，麦肯锡报告截图，主播 PiP 左上角 |
| ~5:45–6:00 | 柱状图（截图引用） | 中国谷子经济市场规模 2017-2029E，橙色柱+折线，主播 PiP 左上角 |
| ~7:15–7:30 | 文字截图 | 国家统计局报告截图，黄色荧光笔高亮关键数字，主播 PiP 右上角 |

**量化结论**：
- 数据可视化出现频率：约 **0.5–0.7 次/分钟**（8.88 分钟内 ~5 次）
- 类型：截图引用为主（约 3 次）；自制大数字 badge（约 1–2 次）
- 平均停留时长（由相邻帧推算）：**约 8–15 秒/次**
- 配色：截图原色（灰白/橙色）；自制 badge 用黑底 + 白字/黄字

### 2.4 B-roll 与插图

| 类型 | 出现次数（估计） | 描述 |
|---|---|---|
| 实拍街景/商场 | ~4–5 次 | POP MART 门店、LABUBU 展台、商场人群、宠物（金毛）、演唱会观众 |
| 产品渲染/3D 动效 | ~2–3 次 | 冰箱产品图（带 ¥5999 大字动效）、电商 App 手机 3D 动画 |
| 数据截图/文件 | ~3–4 次 | 与数据可视化重叠（见 2.3） |
| 主题背景（IP 贴图） | ~2 次 | LABUBU 公仔图案瓦楞纸背景，配话题 badge 章节切换 |
| 电影海报手持 | ~1 次 | 主播手持两张电影海报（作为 prop，非纯 B-roll） |

**B-roll 覆盖率**：35 帧中 13 帧含 B-roll 元素（约 37%）；其中 8 帧为 B-roll + PiP 模式（主播缩至角落）

**PiP 惯例**：
- PiP 出现于几乎所有 B-roll 全屏切镜
- 位置：右上或左上，约 150×130px
- 有描边（白色圆角矩形边框）

### 2.5 人物画面

| 属性 | 观测值 |
|---|---|
| 景别 | 固定**中近景**（肩部以上至腰部），偶有向上移至胸以上 |
| 背景 | 全程**户外咖啡座露台**，自然光，散景虚化；背景一致，无场景切换 |
| 出镜占比 | 主播主镜头约 57%；B-roll 中 PiP 约 20%；合计主播在画面中约 **77%** 时长 |
| 手势 | 高频使用手势辅助表达（指点、摊手、捏拿），活跃度高 |
| 服装 | 职业商务风（深蓝西装外套 + 白衬衫），全片不变 |
| 麦克风 | 胸前夹式无线麦（DJI Mic 2 可见） |

### 2.6 包装元素

**标题条 / 章节 badge**（核心视觉语言，出现约 3–4 次）：

| Badge 类型 | 示例 | 特征 |
|---|---|---|
| 圆角胶囊（大） | 「谷子经济 / Guzi Economy」（紫色）；「"第四消费时代"」（深灰色） | 主标题中文大字 + 英文小字副标题，圆角矩形，带虚线内框描边 |
| 圆角胶囊（小/多） | 「IP 角色」「版本限定」「背后关系」（黑底白字） | 3–4 个小 tag 并排，解释主题关键词 |
| 左上角章节标识条 | 「新逻辑崛起」（金黄色圆角）+ 下方「消费从"证明身份"到"贴合自我"」（深色长条） | 双行，上小下大；出现于 B-roll 帧左上角 |
| 中部横条分割线 | 「第四消费时代」（居中，黑底白字，圆角矩形） | 分割上下 B-roll，字幕落下方区域 |

**关键词动画**（出现约 6–8 次）：

| 类型 | 示例 | 特征 |
|---|---|---|
| 大字浮现（背景透明叠加） | 「传统家电」「为什么？」「消费坐标」 | 字号极大（占画面 1/3–1/2 高），笔刷/手写风，金黄色 |
| 双 badge 并排（概念对比） | 「新消费按钮 / New Consume Button」+ 「关系 / Relationship」 | 一深一浅，中英双语，大小对比 |
| 圆形裁剪 + 主题背景 | 主播视频圆形 crop，背景为 IP 图案 | 章节过渡用，主播变装饰性元素 |

**转场样式**：
- 无明显花哨转场效果；主要为**硬切**（直接切换）
- B-roll 进出：瞬切 + 字幕/badge 叠入动画（淡入/位移）
- 章节切换：全屏主题卡 1–2 秒 → 主播镜头（或反向）

### 2.7 音频与语速

| 指标 | 数值 |
|---|---|
| 语速（字/分钟） | 约 **332 字/分钟**（前 120 秒样本，665 字） |
| 语速（词/分钟，按中文音节估算） | 约 **277 词/分钟** |
| 语言 | 普通话，无英文口播，偶有英文品牌名（LABUBU、POP MART）正常发音 |
| 语调 | 平稳、清晰，略带情绪起伏，非念稿感 |
| BGM/音效 | 本片未分析（需专门音频工具），从帧分析无明显视觉提示 |

## 3. Checklist 摘要（详见 docs/style_checklist.md）

硬指标 15 项（H-01 至 H-15），软指标 12 项（S-01 至 S-12）。

关键量化目标：

```
字幕   : 中文单语 | 位置 Y 900-1050px | 8-16 字/屏 | 白色+黑色描边
图表   : ≥ 0.5 次/分钟 | 柱状图+截图+大数字 pill 混用 | 停留 8-15s
B-roll : ≥ 35% 时长覆盖 | ≥ 60% 的 B-roll 带 PiP
关键词 : ≥ 1 次/分钟大字关键词动画
章节   : 每 2-3 分钟 1 个话题 badge 章节切换
语速   : 280-380 字/分钟（目标值 332）
人物   : 55-70% 主镜头 | 中近景固定 | 单一背景
```

## 4. 对牛初乳视频的适配建议

本参考成片为消费经济科普，牛初乳为健康科普，呈现形式可直接参考，内容层面注意：

1. **图表类型**：牛初乳科普更适合「大数字 badge」（如「IgG 含量 XX mg/ml」）和「对比表格」，截图引用研究报告图表同样适用
2. **主题背景 IP**：需设计牛初乳相关主题背景贴图（替代 LABUBU 图案），建议与产品 logo/奶牛元素结合
3. **B-roll 类型**：实拍 → 实验室/牧场/产品特写；动效 → 分子/细胞动画（可用素材库）
4. **badge 配色**：科普健康类建议用蓝/绿系（信任感），区别于本片的暖金色系
